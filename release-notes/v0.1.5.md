# Sub2API Monitor v0.1.5

This release refreshes the desktop account cards and brings the iOS companion app closer to the current monitor feature set.

## Changes

- Added weekly usage rings to desktop account cards and shared the ring rendering with collapsed views.
- Updated the iOS companion app with current subscription, user usage, OpenAI/Codex usage, theme, and widget behaviors.
- Expanded iOS API models, formatting, JWT scanning, widget snapshots, and simulator test coverage.
- Updated documentation for user usage endpoints and the current iOS companion workflow.

## Notes

- Existing `npm audit` dependency findings remain tracked for a future dependency maintenance release.
- GitHub Actions Node.js 20 runtime deprecation warnings are non-blocking for this release and remain tracked for CI maintenance.
- macOS builds are Apple Silicon artifacts and remain unsigned/unnotarized unless Apple Developer signing secrets are configured. If macOS reports the app is damaged, remove the quarantine attribute or use a locally signed build.
