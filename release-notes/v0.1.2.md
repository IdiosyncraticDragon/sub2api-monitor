# Sub2API Monitor v0.1.2

This maintenance release refreshes the public build with the latest account usage display updates.

## Changes

- Normalized OpenAI account usage display so active account cards and collapsed summaries show usage consistently.
- Improved handling for accounts whose usage data is represented as utilization ratios instead of absolute token totals.
- Added test coverage for the updated API normalization and usage formatting paths.

## Notes

- Existing `npm audit` dependency findings remain tracked for a future dependency maintenance release.
- GitHub Actions Node.js 20 runtime deprecation warnings are non-blocking for this release and remain tracked for CI maintenance.
