# Sub2API Monitor v0.1.1

Maintenance release with desktop fixes and iOS companion app improvements.

## Downloads

| Platform | Asset | Notes |
| --- | --- | --- |
| Windows | `Sub2API.Monitor.Setup.0.1.1.exe` | Recommended Windows installer. |
| Windows | `Sub2API.Monitor.0.1.1.exe` | Portable Windows executable. |
| macOS arm64 | `Sub2API.Monitor-0.1.1-arm64.dmg` | macOS disk image, unsigned unless Apple signing secrets are configured. |
| macOS arm64 | `Sub2API.Monitor-0.1.1-arm64-mac.zip` | Zipped macOS app bundle. |

## Changes

- Fixed repeated login-window loops when stale credentials trigger a 401.
- Improved session-window range display for OpenAI/Codex accounts by deriving a 5h window from reset time when explicit start/end fields are unavailable.
- Added iOS companion app login and usage UI improvements.
- Added iOS WidgetKit snapshot support for active account usage and Today summary.
- Expanded formatting, JWT scanning, usage, view model, and widget snapshot tests.

## Security Notes

- This release does not bundle any Sub2API server address or credential.
- Desktop credentials remain encrypted at rest with Electron `safeStorage`.
- iOS credentials are stored in Keychain.
