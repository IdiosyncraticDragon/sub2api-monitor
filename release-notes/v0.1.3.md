# Sub2API Monitor v0.1.3

This maintenance release refreshes the public build with refined collapsed progress indicators.

## Changes

- Updated collapsed ring indicators to show both 5-hour session usage and 7-day usage.
- Refined collapsed segmented bars and spotlight view so short-term and weekly usage are easier to compare.
- Added component test coverage for the updated collapsed usage indicators.

## Notes

- Existing `npm audit` dependency findings remain tracked for a future dependency maintenance release.
- GitHub Actions Node.js 20 runtime deprecation warnings are non-blocking for this release and remain tracked for CI maintenance.
