# Sub2API Monitor v0.1.4

This release adds user usage monitoring and refreshes the collapsed active-account summary.

## Changes

- Added a user usage monitor view for reviewing user-level usage totals.
- Added API, preload, renderer hook, and component support for user usage data.
- Fixed collapsed active account counting so the compact view reflects active accounts more accurately.
- Added test coverage for user usage loading and collapsed count behavior.

## Notes

- Existing `npm audit` dependency findings remain tracked for a future dependency maintenance release.
- GitHub Actions Node.js 20 runtime deprecation warnings are non-blocking for this release and remain tracked for CI maintenance.
- macOS builds are Apple Silicon artifacts and remain unsigned/unnotarized unless Apple Developer signing secrets are configured.
