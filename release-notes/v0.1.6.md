# Sub2API Monitor v0.1.6

This release improves authentication resilience by refreshing expired access tokens automatically when a refresh token is available.

## Changes

- Added automatic access-token refresh before polling or fetching cached views when the stored access token expires.
- Added API support for refresh-token exchange and related authentication error handling.
- Updated authentication and API tests to cover refresh success, refresh failure, and fallback re-login behavior.
- Documented the refresh-token endpoint behavior.

## Notes

- Existing `npm audit` dependency findings remain tracked for a future dependency maintenance release.
- GitHub Actions Node.js 20 runtime deprecation warnings are non-blocking for this release and remain tracked for CI maintenance.
- macOS builds are Apple Silicon artifacts and remain unsigned/unnotarized unless Apple Developer signing secrets are configured. If macOS reports the app is damaged, remove the quarantine attribute or use a locally signed build.
