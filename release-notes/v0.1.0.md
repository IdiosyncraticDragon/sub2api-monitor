# Sub2API Monitor v0.1.0

First public snapshot release.

## Downloads

| Platform | Asset | Notes |
| --- | --- | --- |
| Windows | `Sub2API Monitor Setup 0.1.0.exe` | Recommended installer for Windows. |
| Windows | `Sub2API Monitor 0.1.0.exe` | Portable executable. |
| macOS | `Sub2API Monitor-0.1.0.dmg` | macOS installer image, produced by the GitHub Release workflow on macOS. |
| macOS | `Sub2API Monitor-0.1.0-mac.zip` | Zipped macOS app bundle, produced by the GitHub Release workflow on macOS. |

## Highlights

- Frameless always-on-top monitor window for active Sub2API accounts.
- First-run server setup and persistent login through the configured Sub2API admin site.
- Encrypted token persistence via Electron `safeStorage`.
- Grouped account list, dashboard totals, tray controls, theme settings, and collapsed mini mode.

## Security Notes

- The app does not include any bundled Sub2API server address or credential.
- JWT credentials are encrypted at rest with the platform credential backend.
- macOS artifacts from the default open-source workflow are unsigned unless Apple signing and notarization secrets are configured.
